# SayUpAI 모바일 앱 디자인 계획

## 브랜드 아이덴티티

SayUpAI는 AI 발표 코치 서비스로, 다크 테마와 시안/청록색 강조색을 사용하는 고급스러운 테크 스타일을 유지합니다. 모바일 앱은 sayupai.co.kr 웹사이트의 디자인을 100% 재현합니다.

## 화면 목록

### 1. 홈 화면 (HomeScreen)
- SayUpAI 로고 + 마이크 아이콘 (상단 좌)
- "샘플 보기" 버튼 (상단 우)
- "AI 발표 코치" 태그 (시안 테두리)
- 히어로 텍스트: "발표가 두렵다면 AI에게 맡기세요"
- 통계 3개 (97%, 3분, 0원) - 시안 테두리 카드
- CTA 버튼 3개 (시작하기, 실시간 스트리밍, 게스트 체험)
- 하단 안내 텍스트

### 2. 계정 화면 (AccountScreen)
- 사용자 아바타 (이니셜) + Free/Pro 배지
- 현재 플랜 카드
- "Pro로 업그레이드" 버튼
- 플랜 비교 버튼
- 초대 코드 섹션
- 계정 관리 (히스토리 관리, 회원 탈퇴)

### 3. 가격/구독 화면 (PricingScreen)
- 월간/연간 토글
- Free 플랜 카드
- Pro 플랜 카드 (강조)
- 기능 비교 테이블
- 초대 코드 입력

### 4. 실시간 코칭 화면 (LiveCoachScreen)
- 카메라 뷰파인더
- 연결 상태 패널
- AI 코칭 피드
- 시작/종료 버튼

### 5. 발표 성장 대시보드 (HistoryScreen)
- 통계 카드 3개
- 필터 탭 (전체/완료/실패)
- 목적 필터 (IR 피치, 팀 보고 등)
- 분석 기록 목록

### 6. 분석 시작 화면 (StartAnalysisScreen)
- 영상 업로드 영역
- 발표 목적 선택
- 분석 시작 버튼

### 7. 분석 결과 화면 (AnalysisResultScreen)
- 발표 점수 (0-100)
- 말하기 습관 분석
- 핵심 메시지
- 다음 미션
- 공유/저장 버튼

### 8. 로그인/회원가입 화면 (AuthScreen)
- SayUpAI 로고
- 소셜 로그인 버튼
- 이메일 로그인

## 핵심 사용자 플로우

**신규 사용자 플로우:** 홈 → "시작하기" → 회원가입 → 분석 시작 → 영상 업로드 → 분석 결과

**실시간 코칭 플로우:** 홈 → "실시간 스트리밍" → 카메라 권한 → 코칭 시작 → 피드백 수신

**구독 업그레이드 플로우:** 계정 → "Pro 업그레이드" → 가격 화면 → 인앱결제 → 완료

**게스트 체험 플로우:** 홈 → "게스트로 샘플 체험" → 샘플 리포트 확인

## 탭바 구조

하단 탭바 2개:
- 홈 (house 아이콘)
- 계정 (person 아이콘)

## AdMob 광고 배치

- 홈 화면 하단: 배너 광고
- 분석 결과 화면 상단: 배너 광고
- 분석 완료 후: 전면 광고 (인터스티셜)
- 추가 분석 횟수 획득: 보상형 광고

- Pro 월간: ₩19,000/월
- Pro 연간: ₩190,000/년 (17% 할인)
- 플랫폼: iOS App Store + Google Play Store
